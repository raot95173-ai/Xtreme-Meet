# Xtreme Meetings

Xtreme Meetings is a secure, open-source video meeting app like Zoom with unlimited meeting time.

## Features
- Unlimited meetings
- WebRTC video/audio
- Secure & private
- Free & open source

## Installation
git clone https://github.com/raot95173-ai/Xtreme-Meet.git
cd Xtreme-Meet/server
npm install
npm start

Open: http://localhost:3000
